*Arduino based project*

What is used :
-a spotlight (8 volt)
-Arduino (example: Uno, Nano)
-UltraSonic sensor HC-SR04
-LED WS2812B (6 LED)
-Relay
-microphone (KY-037)
-battery 9V


How Does it work:
when an object pass through the ultrasonic sensor, the spotlight turn on, only if the spotlight is on the LEDs light up when the microphone detects any sound.
